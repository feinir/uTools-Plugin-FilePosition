快速定位文件和文件夹

项目：
https://github.com/feinir/uTools-Plugin-FilePosition


类似Wox中的快速定位到文件功能。

在uTools中输入 go 空格
进入后，可以直接输入路径 C:\ ， 然后就会一层一层的提示选择进入文件。最后打开，也可直接一次输入所有路径。
